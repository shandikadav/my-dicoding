Berita Desa

Kerja Bakti Perawatan Selokan di Wilayah Sekitar Rumah

Kerja bakti merupakan kegiatan gotong royong yang dilakukan oleh masyarakat desa. Kegiatan ini dilakukan secara bersama-sama untuk membersihkan lingkungan desa, memperbaiki infrastruktur desa, dan memperindah taman-taman desa. Kerja bakti dilakukan secara rutin setiap bulan, dan diikuti oleh seluruh warga desa.

Masyarakat Desa Diminta Untuk Suntik Vaksin

Pemerintah desa mengimbau seluruh masyarakat desa untuk melakukan suntik vaksin. Hal ini dilakukan sebagai upaya pencegahan penularan virus Covid-19. Suntik vaksin dilakukan secara gratis di puskesmas desa, dan diikuti oleh seluruh warga desa.
